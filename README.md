# Hanko - Electronic Signature Tool

A tool for electronically signing documents.

## Demo
[Hanko - Electronic Signature Tool](https://code4fukui.github.io/hanko/)

## Features
- Generate public and private keys
- Issue and manage digital certificates
- Electronically sign documents
- Verify signed documents

## Requirements
None

## Usage
1. Generate public and private keys.
2. Obtain a digital certificate for the public key.
3. Register the digital certificate.
4. Use the private key to electronically sign documents.
5. Verify signed documents using the public key and digital certificate.

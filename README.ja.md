# Hanko - 電子署名ツール

Hankoは、文書に電子署名を行うためのツールです。公開鍵と秘密鍵の生成、電子証明書の発行と登録、電子署名の生成と検証といった機能を提供します。

## デモ
[Hanko - 電子署名ツール](https://code4fukui.github.io/hanko/)

## 機能
- 公開鍵と秘密鍵の生成
- 電子証明書の発行と登録 
- 文書への電子署名と検証

## 使い方
1. 公開鍵と秘密鍵を生成する
2. 電子証明書を発行する
3. 電子証明書を登録する
4. 文書に電子署名する
5. 電子署名を検証する

## データ・API
このプロジェクトでは以下のライブラリを使用しています:
- [sec.js](https://github.com/code4fukui/sec.js)
- [Base32](https://github.com/code4fukui/Base32)
- [input-text](https://github.com/code4fukui/input-text)
- [qr-code](https://github.com/code4fukui/qr-code)
